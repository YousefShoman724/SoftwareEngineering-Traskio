TASK MANAGEMENT TEST CASES — Traskio Project

=====================================
TASK CREATION TEST CASES
=====================================

Test Case ID: TM-01
Title: Add New Task with Valid Data
Description: Verify that a logged-in user can create a new task with valid information.
Precondition: User is logged in.

Steps:
1. Open the dashboard.
2. Click on "Add Task".
3. Enter a task title.
4. Enter a task description.
5. Select priority.
6. Set deadline (optional).
7. Click Save.

Expected Result:
- Task is created successfully.
- Task appears in the task list.


Test Case ID: TM-02
Title: Add Task with Empty Title
Description: Verify task creation fails when title is empty.
Precondition: User is logged in.

Steps:
1. Open dashboard.
2. Click Add Task.
3. Leave task title empty.
4. Click Save.

Expected Result:
- Task is not created.
- Validation error message is shown.
